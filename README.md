# Homebrew Formulae

> A Homebrew tap for my personal projects.

## Usage

First, tap this repository:

```
brew tap jez/formulae
```

Then, install software like normal:

```
brew install bask

brew install pandoc-sidenote

brew install stargaze

brew install git-heatmap
```
